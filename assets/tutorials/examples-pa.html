Sequence / BASICS / Simple Message

ਇਹ ਉਦਾਹਰਨ “A ਵੱਲੋਂ B ਨੂੰ ਸੁਨੇਹਾ” ਦਿਖਾਉਂਦੀ ਹੈ।

ਕਿਉਂ ਵਰਤੋ

  • ਸੰਦੇਸ਼ਾਂ ਦਾ ਕ੍ਰਮ ਸਪਸ਼ਟ ਹੁੰਦਾ ਹੈ।
  • “ਕੌਣ ਪਹਿਲਾਂ ਬੋਲਿਆ” ਆਸਾਨੀ ਨਾਲ ਸਮਝ ਆਉਂਦਾ ਹੈ।

ਕਿਵੇਂ ਵਰਤੋ

  1. @startuml ਨਾਲ ਸ਼ੁਰੂ ਕਰੋ ਅਤੇ @enduml ਨਾਲ ਖਤਮ ਕਰੋ।
  2. Alice -> Bob: Message ਲਿਖੋ।

ਉਦਾਹਰਨ ਵਰਤੋਂ

  • ਲਾਗਇਨ ਫਲੋ: User → System.
  • ਚੈਟਬੌਟ ਗੱਲਬਾਤ।

ਅਭਿਆਸ

  1. ਨਾਂ ਬਦਲ ਕੇ ਆਪਣੀ ਗੱਲਬਾਤ ਲਿਖੋ।
  2. participant ਲਾਈਨ ਜੋੜ ਕੇ ਦੋ ਸੁਨੇਹੇ ਲਿਖੋ।
  3. ਦੋ ਲਾਈਨਾਂ ਵਿੱਚ ਵੰਡੋ।
  4. ਪਹਿਲਾ ਬੋਲਣ ਵਾਲਾ ਬਦਲੋ।
  5. ਤੀਜਾ ਕਿਰਦਾਰ ਜੋੜੋ।

PlantUML ਨਿਯਮ

  • @startuml ਤੋਂ @enduml ਤੱਕ।
  • A -> B ਕਾਲ, A --> B ਜਵਾਬ।
  • ਕੋਲਨ ਤੋਂ ਬਾਅਦ ਲਿਖਤ।
  • ਸਮਾਂ ਉੱਪਰ ਤੋਂ ਹੇਠਾਂ।
  • ਲਿਖਤ ਵਿਕਲਪਿਕ।
  • ਸਪੇਸ ਵਾਲੇ ਨਾਂ quotes ਵਿੱਚ।

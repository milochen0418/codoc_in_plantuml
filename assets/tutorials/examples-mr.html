Sequence / BASICS / Simple Message

हे उदाहरण “A कडून B कडे संदेश” दाखवते.

का वापरावे

  • संवादाचा क्रम स्पष्ट होतो.
  • “कोण आधी बोलतो” हे कळते.

कसे वापरावे

  1. @startuml ने सुरू करून @enduml ने संपवा.
  2. Alice -> Bob: Message असे लिहा.

उदाहरण उपयोग

  • लॉगिन फ्लो: User → System.
  • चॅटबॉट संवाद.

सराव

  1. नावे बदलून रोजच्या संभाषणाचे उदाहरण लिहा.
  2. participant जोडून दोन संदेश लिहा.
  3. दोन ओळींमध्ये विभागा.
  4. पहिला बोलणारा बदला.
  5. तिसरा पात्र जोडा.

PlantUML नियम

  • @startuml ते @enduml आवश्यक.
  • A -> B कॉल, A --> B उत्तर.
  • कोलन नंतर मजकूर.
  • वेळ वरून खाली.
  • मजकूर ऐच्छिक.
  • स्पेस असलेल्या नावांना quotes.
